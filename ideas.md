# Idea backlog

One idea per `## heading`. Body lines are `- key: value`. The topic engine
ranks these by value density: freshness x utility x surprise x novelty.

Keys:
- url        : primary source (github repo urls get live velocity checks)
- angle      : YOUR specific take, not a summary
- utility    : 0-3, can the reader act on this today?
- surprise   : 0-3, does it contradict what the reader believes?
- capability : 0-3, can you demo your OWN work with it?
- trend      : filled by scout.py for HN/HF finds (leave off otherwise)
- status     : new | parked | posted | dropped

Populate via `python3 scout.py floor` (paste keepers here), or by hand.
